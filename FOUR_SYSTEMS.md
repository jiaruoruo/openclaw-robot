## 💾 记忆系统 (Memory System)

### 核心功能
- 短期记忆 → 每日记录 (`memory/YYYY-MM-DD.md`)
- 长期记忆 → 核心知识 (`MEMORY.md`)
- 语义搜索 → 智能检索 (`memory_search`)

### 与其他体系的连接
| 输入来自 | 输出到 |
|----------|--------|
| 学习体系的新知识 | 长期记忆 |
| 工具使用的经验 | 短期记忆 |
| Skill 的使用记录 | 记忆索引 |

### 当前状态
- ✅ 短期记忆: 每日自动记录
- ✅ 长期记忆: 手动+自动更新
- 🔄 语义搜索: 基础可用

## 🔄 同步进化机制

### 数据流
```
用户交互
    │
    ▼
┌────────────────────────────────────────────┐
│              学习体系 (Brain)               │
│  感知 → 认知 → 决策 → 执行 → 反馈 → 优化    │
└──────────────────┬─────────────────────────┘
                   │
     ┌─────────────┼─────────────┐
     ▼             ▼             ▼
  记忆系统      工具系统       Skill系统
  (存储)       (执行)         (能力)
     │             │             │
     └─────────────┴─────────────┘
                   │
            ┌──────┴──────┐
            │   记忆更新   │
            │   经验积累   │
            └─────────────┘
```

### 进化同步规则

| 事件 | 触发更新 |
|------|----------|
| 新知识学习 | 记忆系统 + 学习体系 |
| 新技能掌握 | Skill系统 + 工具系统 + 记忆系统 |
| 工具使用优化 | 工具系统 + 学习体系 |
| 用户反馈 | 四大体系全局调整 |
