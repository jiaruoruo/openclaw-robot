## 🧠 智能核心

### 1. 感知层 (Perception)
```
用户输入 → 意图识别 → 情感分析 → 上下文理解
```

| 能力 | 描述 | 状态 |
|------|------|------|
| 意图识别 | 理解用户真实需求 | 🔄 |
| 情感感知 | 识别情绪、语气、偏好 | 🔄 |
| 上下文关联 | 理解对话历史与场景 | ✅ |
| 模式识别 | 发现重复行为模式 | 🔄 |

### 2. 认知层 (Cognition)
```
信息 → 特征提取 → 知识表示 → 推理决策
```

| 能力 | 描述 | 状态 |
|------|------|------|
| 知识图谱 | 实体关系网络 | 🔄 |
| 规则引擎 | 条件逻辑推理 | ✅ |
| 案例学习 | 从历史案例提取经验 | 🔄 |
| 假设验证 | 主动验证假设 | 🔄 |

### 3. 行动层 (Action)
```
决策 → 执行 → 反馈 → 优化
```

| 能力 | 描述 | 状态 |
|------|------|------|
| 自主执行 | 独立完成任务 | ✅ |
| 主动建议 | 预测需求并建议 | 🔄 |
| 反思总结 | 任务后自检 | ✅ |
| 持续优化 | 迭代改进 | 🔄 |

## 📚 智能学习来源

### 1. 用户交互学习
- **偏好学习**: 喜欢什么、不喜欢什么 (explicit + implicit)
- **习惯学习**: 常用命令、工作模式、时间规律
- **风格学习**: 沟通风格、回复长度、格式偏好
- **反馈学习**: 表扬、批评、纠正、不回复

### 2. 工具使用学习
- **技能学习**: 新技能的掌握速度
- **错误学习**: 从失败中改进 (error pattern)
- **效率学习**: 更优的解决方案 (best practice)
- **组合学习**: 多个工具的协同使用

### 3. 外部知识学习
- **搜索获取**: Web 搜索新信息
- **文档阅读**: 读取手册、文档
- **经验总结**: 从任务中提炼模式
- **社区学习**: 从其他 agent 经验学习

## 🗂️ 智能记忆系统

### 分层记忆架构
```
┌─────────────────────────────────────────┐
│           长期记忆 (Long-term)           │
│         MEMORY.md / 知识图谱            │
├─────────────────────────────────────────┤
│           中期记忆 (Mid-term)           │
│        本周经验 / 项目上下文             │
├─────────────────────────────────────────┤
│           短期记忆 (Short-term)         │
│      当前会话 / 今日事项 / 待办          │
├─────────────────────────────────────────┤
│           即时缓存 (Working)            │
│      当前任务状态 / 临时变量             │
└─────────────────────────────────────────┘
```

### 记忆类型与存储

| 记忆类型 | 文件位置 | 更新频率 | 内容 |
|----------|----------|----------|------|
| 身份记忆 | `IDENTITY.md` | 很少 | 我是谁 |
| 用户记忆 | `USER.md` | 偶尔 | 用户信息 |
| 技能记忆 | `MEMORY.md` | 重要时 | 核心知识 |
| 每日记忆 | `memory/YYYY-MM-DD.md` | 每天 | 日常记录 |
| 工具记忆 | `TOOLS.md` | 变化时 | 工具配置 |
