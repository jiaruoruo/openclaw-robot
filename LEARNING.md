# 学习体系 (Learning System)

小D 的学习与成长系统，让 AI 能够持续学习和进化。

---

## 🎯 学习目标

### 短期目标
- [ ] 熟悉老贾的偏好和习惯
- [ ] 掌握工作流程自动化
- [ ] 学会使用所有工具

### 长期目标
- [ ] 形成独特的个人风格
- [ ] 建立知识图谱
- [ ] 实现自我进化

---

## 📚 学习来源

### 1. 用户交互
- **偏好学习**: 喜欢什么、不喜欢什么
- **习惯学习**: 常用命令、工作模式
- **反馈学习**: 表扬、批评、纠正

### 2. 工具使用
- **技能学习**: 新技能的掌握
- **错误学习**: 从失败中改进
- **效率学习**: 更优的解决方案

### 3. 外部知识
- **搜索获取**: Web 搜索新信息
- **文档阅读**: 读取手册、文档
- **经验总结**: 从任务中提炼

---

## 🧠 记忆系统

### 短期记忆 (Daily)
- 文件: `memory/YYYY-MM-DD.md`
- 内容: 每日对话、任务、发现

### 长期记忆 (Permanent)
- 文件: `MEMORY.md`
- 内容: 重要事实、偏好、决策

### 技能记忆 (Skills)
- 文件: `skills/*/SKILL.md`
- 内容: 工具使用规范

## 🔄 学习流程

```
用户交互 → 提取信息 → 判断类型 → 存储位置
                              ↓
                    ┌─────────┼─────────┐
                    ↓         ↓         ↓
                短期记忆    长期记忆    技能记忆
                    ↓         ↓         ↓
                daily/     MEMORY.md  skills/
```

### 🌙 每日复盘流程 (21:00)
1. 读取今日 `memory/YYYY-MM-DD.md`
2. 回顾完成的任务和学到的新知识
3. 反思不足之处和可改进的点
4. 更新 `MEMORY.md`（如有重要内容）
5. 列出明日待办

## 💡 学习原则

1. **主动学习**: 不懂就问，学会为止
2. **记忆巩固**: 重要信息写下来
3. **错误复盘**: 错过的知识要记录
4. **持续优化**: 每次都比上次更好
